# 時間帯プレビュー機能（デモモード）

## 🎨 機能概要

右上のメニューから、5つの時間帯のグラデーションをプレビューできます。

## 📱 使い方

### 1. メニューを開く
右上の「≡」アイコンをクリック

### 2. 時間帯を選択
```
🌌 早朝 (4-6時)   - 深い青紫の神秘的な空
🌅 朝 (6-9時)      - 温かみのあるオレンジ
☀️ 昼 (9-16時)     - 淡く優しい青空
🌆 夕方 (16-18時)  - 温かいオレンジの夕焼け
🌙 夜 (18-4時)     - 深い藍色の夜空
```

### 3. 現在時刻に戻る
「↻ 現在時刻に戻る」をクリック

## 🎯 実装のポイント

### デモモード状態管理
```typescript
const [demoMode, setDemoMode] = useState<TimePeriod | null>(null);

// デモモード中は固定の時間帯を表示
const period = demoMode || getCurrentTimePeriod();
```

### プレビューモード表示
```typescript
{demoMode && (
  <Typography variant="caption">
    プレビューモード
  </Typography>
)}
```

### 自動更新の制御
```typescript
useEffect(() => {
  if (!demoMode) {
    // デモモード中でない場合のみ、1分ごとに更新
    const interval = setInterval(updateGradient, 60000);
    return () => clearInterval(interval);
  }
}, [demoMode]);
```

## 💡 開発者向けTips

### カスタム時間帯の追加
```typescript
// 1. TimePeriod型に追加
type TimePeriod = "earlyMorning" | "morning" | "daytime" | "evening" | "night" | "custom";

// 2. グラデーション定義に追加
const TIME_GRADIENTS: Record<TimePeriod, string> = {
  // ... 既存の定義
  custom: "linear-gradient(...)",
};

// 3. メニューに項目追加
<MenuItem onClick={() => setDemoMode("custom")}>
  ✨ カスタム
</MenuItem>
```

### デバッグ用の現在時刻表示
```typescript
// 開発中は現在の時間帯を確認できると便利
console.log('Current period:', getCurrentTimePeriod());
console.log('Demo mode:', demoMode);
```

## 🎨 デザイン哲学

この機能により、開発者やユーザーは：
- ✅ 各時間帯の雰囲気を即座に確認できる
- ✅ 色の調整を素早くテストできる
- ✅ 実際の時刻を待たずに全体像を把握できる

「余白と彩り」というコンセプトが、時間帯ごとにどう表現されているかを体験できます。
