# Skyle 開発 12 日目

## 実装内容

### 配色の設計

自分で色を選び、グラデーションを作成

#### 朝焼け（確定：バージョン H）

```css
background: linear-gradient(
  to bottom,
  #a8c5f0 0%,
  #b8d0f3 45%,
  #f0f8ff 65%,
  #fffff0 82%,
  #ffe4e1 94%,
  #ffc4b8 100%
);
```

- 青 65% / 白〜黄 23% / ピンク〜橙 12%
- 「透き通った水色からピンク、オレンジ」のイメージ

#### 冬の昼（確定：バージョン P）

```css
background: linear-gradient(
  to bottom,
  #2850a8 0%,
  #5078c8 5%,
  #7ca0dc 15%,
  #a8c8f0 35%,
  #c8def5 55%,
  #e8f2fa 80%,
  #ffffff 100%
);
```

- 濃い青を上部 5%に絞ることで透明感を出す
- 「澄んだ青にほわっとした雲」のイメージ

## 学んだこと

### 透明感の出し方

- 「白地にセロファンを重ねた」感覚
- 濃い色の「面積」を絞ると透明感が生まれる
- グラデーションの配分が重要

### デザイン判断

- 「見た印象」と「背景としての適切さ」は分けて考える
- 本物の再現より「ユーザーにとって心地いいか」で判断
